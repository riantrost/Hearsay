```
Member     { id, campaignId, name, role: owner|player, status: active|pending }
Pin        { id, campaignId, x, y, name, hidden?, description?, sealed? }
Event      { id, pinId, createdAt, canonLine, atmosphere?, participantIds }
Testimony  { id, eventId, memberId, createdAt, text, markText? }
Bounty     { id, campaignId, postedBy, target, reason, postedAt, status, struckAt? }
```
**Mark folds into `markText` on Testimony** (a highlight, not a content type). There is no global clock: `createdAt` epoch-ms stamps carry ordering, each pin's own event history is its grace-window clock, and recency illumination derives from the campaign's newest event (never wall-clock now).
